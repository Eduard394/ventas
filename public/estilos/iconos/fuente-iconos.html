¿Cómo usar la fuente de iconos?

Para usar la fuente de íconos se debe incluir el código html.

<i class="ib-iconos ib-antena"></i>

Siempre lleva la clase ib-iconos.

La clase ib-antena cambia según el ícono que se desee usar.
